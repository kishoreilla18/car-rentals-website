# Assume df is your DataFrame

This code splits the dataset into training and testing sets using scikit-learn's train_test_split. The 'treatment' column is used as the target variable (label), and all other columns serve as features. 70% of the data is used for training and 30% for testing, ensuring reproducibility with a fixed random seed. The resulting split shows that the training set contains 881 samples and the test set contains 378 samples, each with 12 features.
